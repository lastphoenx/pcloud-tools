# pcloud-tools
Python-CLI-Tools für Backups, Admin &amp; Sync über die offizielle pCloud-API.

## Abhängigkeiten

Alle Skripte verwenden ausschließlich Module aus der Python-Standardbibliothek.
Für den Komfort beim Einlesen von `.env`-Dateien kann zusätzlich
[`python-dotenv`](https://pypi.org/project/python-dotenv/) installiert werden;
das Paket wird in `pcloud_binary_client.py` optional importiert und ist in
`requirements.txt` hinterlegt.
